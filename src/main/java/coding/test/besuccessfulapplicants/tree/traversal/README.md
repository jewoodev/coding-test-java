## 문제 설명

이진 트리를 표현한 배열 nodes를 인자로 받는다. 예를 들어 nodes가 [1, 2, 3, 4, 5, 6, 7]이면 다음과 같은 트리를 표현한 것이다. 해당 이진 트리에 대해 전위 순회, 중위 순회, 후위 순회 결과를 반환하는 solution() 함수를 구현하라.

### 제약 조건
- 입력 노드값의 개수는 1개 이상 1000개 이하다.
- 노드값은 정수형이며, 중복되지 않는다.

### 입출력의 예

| nodes                 | return                                              |
|-----------------------|-----------------------------------------------------|
| [1, 2, 3, 4, 5, 6, 7] | ["1 2 4 5 3 6 7", "4 2 5 1 6 3 7", "4 5 2 6 7 3 1"] |
